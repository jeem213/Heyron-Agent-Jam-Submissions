# Memory Tags Guide

## Standard Tags

Use these tags in ALL memory entries:

### Priority Tags
- `#important` - Key decisions, milestones, preferences
- `#routine` - Regular updates, casual check-ins
- `#decision` - Explicit decisions made
- `#task` - Any follow-up tasks created

### Category Tags
- `#session` - General conversation
- `#business` - AI Agent Builder, money, startup
- `#personal` - Family, hobbies, health
- `#technical` - Triggers, setup, integrations
- `#system` - OpenClaw configuration

### Topic Tags
- `#skills` - Skill-related entries
- `#rules` - Rule changes
- `#memory` - Memory system updates
- `#product` - Stuart in a Box product
- `#research` - Research findings

### Private Tags (Use Sparingly)
- `#private` - Sensitive business info
- `- removed from public version` - TOP SECRET research

## Tagging Rules

1. EVERY entry needs at least one priority tag
2. Use category tags to group by type
3. Use topic tags for specific subjects
4. Keep tags lowercase, no spaces

## Example

```markdown
# Memory: May 2, 2026

## Decision: Added new rules
#decision #important #rules

## Task: Update skills
#task #important #skills
```
