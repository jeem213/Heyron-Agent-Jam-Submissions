# Remember - Save to Memory

One word saves the entire conversation to memory files and GitHub.

## Steps

1. **Create memory file** - `memory/YYYY-MM-DD.md`
2. **Add session info** - Decisions, topics, tasks, notes
3. **Add tags** - #session #topic etc
4. **GitHub sync** - Pull + push to backup
5. **Confirm** - Tell user what's saved

## Example

"remember" → Saved to memory + GitHub!
